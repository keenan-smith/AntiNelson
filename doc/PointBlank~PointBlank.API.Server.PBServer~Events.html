PBServer Events

The events of the PBServer class are listed below. For a complete list of PBServer class members, see the PBServer Members topic.

Public Static Events

OnConsoleInput Gets called when a command is entered into the console.
OnConsoleOutput Gets called when the console outputs text.
OnPlayerJoin Gets called when a player joins.
OnPlayerLeave Gets called when a player leaves.
